HTML ->홈페이지를 만들어 달라고 컴퓨터에 요청하는 언어
사람과 웹브라우저 간의 약속
웹페이지가 어떻게 생겼고 어떤 정보를 가지고 있어야 하는지를 웹브라우저에 명령할때 사용하는 언어

HTML의 문법 syntax

1) 태그 tag - HTML의 가장 중요한 핵심
2) 속성 

윈도우 웹 페이지 만드는 방법 - 메모장 - 아무 문장이나 써놓고 - 파일 - 저장 - 바탕화면에 새로운 폴더 하나 만들고 - 파일 이름을 a.html로 - 파일형식을 모든 파일로 - 인코딩을 UTF-8로 하고 저장

a.html에서 a는 파일 이름, html은 확장자 - 확장자가 html인 것은 html파일이라는 것이고, 파일을 더블클릭하면 웹브라우저가 실행될 것인데, 이는 확장자가 html이기 때문임


markup과 관련된 것이 tag임.

*모든 tag 알기

tag는 다양한 기능을 추가해줌.
시작태그와 닫히는 태그가 있음
예시 : 

웹브라우저 상에서 강조하고 싶은 부분에 <strong>강조하고자하는 부분</strong> 이렇게 입력하면 웹브라우저 상에 진한 글씨로 표시됨 
<strong>:뒤따르는 문자를 진하게 표시하라 </strong>전까지
 /가 없는 태그 - 시작태그
 /가 있는 태그 - 닫히는 태그

<h1>과 </h1> - heading 1의 약자 - 대제목으로 처리(굵고 큰 문자로, 줄 바꿈을 하며)
소제목은 <h2>과 </h2>

속성이라는 문법과 링크라는 기능 - html에서 가장 중요한 것
인물 이름에 링크를 걸려면 <a href="복사한 링크주소 입력">인물 이름</a>
a태그는 anchor의 약자 - 링크가 걸린 문자는 배의 닻 과 같은 기능을 함. 비유를 하자면 정박한 배의 닻

새로운 탭이 열리면서 페이지 이동하고 싶다면 "복사한 링크주소" 뒤에 target="_blank"입력(사실 위치는 상관 없음) title="내용" 추가 입력하면 마우스 커서를 링크걸린 곳 위에 두었을 때 간단한 설명이 뜨게 하는것

<li> </li> : list tag 목록 태그
새로운 html파일 만들고 - 텍스트 목록 만들기 - 각각 항목들 마다 위의 태그로 감싸주기 
<ul> </ul> 태그 : unordered list - grouping(다른 목록과는 분리. li태그안에서만 유용. li태그 문단 윗줄에 <ul> 아랫줄에 </ul>써서 다른 그룹과 분리)
<ol> </ol> : ordered list - grouping할때 순서화시킴.(1. 2. 3. 4. 로 번호매김)


문서의 구조
앞서 li태그 때 만든 b.html을 이름만 c.html로 바꿈(주소에서도)
밑의 목록은 삭제하고 - 필요한 내용 복사해서 붙여넣기 - 큰제목 <h1>태그 이용해서 만들고 - <h2>로 소제목 - <title></title>태그로 탭의 제목 바꾸기 - 글씨가 깨지는 경우는 <meta char set="utf-8">태그로 해결할 수 있음

<title>, <meta char>태그는 앞서 배운 태그들과 성격이 다름. 앞서 배운 태그들은 웹페이지의 본문(웹의 하얀 부분에 표시되는)에 해당되는 것임. 그러나 <title>, <meta~> 본문에 해당하지는 않지만 부가적으로 꾸며주는 정보라고 할 수 있음
본문이 아닌 태그는 <head>태그 안에 들어오고 본문인 태그는 <body>태그 안에 들어옴

<html>태그가 가장 큰 포용? 가장 바깥쪽에 두고 그 안에 <head>, <body>태그 - 이 태그들 안에 <title> / <h1> 등등 태그를 둠

DOCTYPE : Document type declaration 
어떤 html코드를 작성할 때 자신이 작성한 html코드가 어떤 방식의 코드로 작성되었는지(어떤 약속에 따라서 제정된 것인지) 브라우저에게 알려주는 문자
<!DOCTYPE html>을 <html>태그 위에 그냥 기계적으로,,,,넣기,,


홈페이지 만들기
5개의 파일이 필요 - c.html의 내용 그대로 재활용 - c.html을 다른이름으로 저장 (파일이름은 index.html) 모든파일로 저장 - 링크 먼저 마련(각 요소마다 ex. HTML, 기술소개, 기본문법 등등) 예시) <href="1.html"> <href="index.html">를 <h1>등의 오른쪽에 입력 - index.html을 다른이름으로 저장 1.html로 - <h2>기술 소개</h2> 아래 줄에 내용 입력 - <h2> </h2>사이의 제목과 그 아랫줄의 내용을 바꿔주면서 이 과정을 반복 -
